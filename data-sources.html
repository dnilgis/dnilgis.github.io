Data Sources

Full transparency on where every number on AGSIST comes from.

Commodity Prices

📊 Yahoo Finance (via yfinance)

Used for: Corn, soybean, wheat, live cattle, lean hog, crude oil, natural gas, gold, silver, Dollar Index, 10-yr Treasury futures, and cryptocurrency prices.

Delay: 15 minutes minimum during market hours.

Update frequency: Every 30 minutes via GitHub Actions during weekday trading hours.

finance.yahoo.com — free market data via the yfinance Python library.

Weather

🌤 Open-Meteo

Used for: Hourly temperature, wind speed, wind direction, humidity, precipitation probability, and derived spray/urea risk calculations.

Update frequency: Every hour.

open-meteo.com — free, open-source weather API.

🌧 NOAA / National Weather Service

Used for: Radar imagery via Windy.com embed.

weather.gov

🗺 NOAA Drought Monitor

Used for: Weekly U.S. Drought Monitor map embed.

droughtmonitor.unl.edu

Prediction Markets

🎯 Kalshi & Polymarket

Used for: Agricultural and economic event probabilities on the Ag Odds page. Data fetched server-side via GitHub Actions from Kalshi's trading API and Polymarket's Gamma API.

kalshi.com · polymarket.com

USDA Reports

🌾 USDA APIs

Used for: USDA report calendar (WASDE, Crop Progress, Grain Stocks, Acreage, Cattle on Feed) via the USDA Economic Research Service and NASS APIs. Report dates extended through October 2026.

usda.gov — public APIs, no key required.

Geocoding

📍 Nominatim / OpenStreetMap

Used for: Reverse geocoding browser lat/lng to city, state, and ZIP code. No personal location data is stored on our servers.

Farmers First Ag Index (FFAI)

📈 Farmers First Agri Service — Proprietary

The FFAI is a composite index calculated by Farmers First Agri Service from publicly available agricultural economic data. Available at farmers1st.com.

Cash Bids

🚜 Barchart OnDemand

Local elevator cash bids are sourced from the Barchart OnDemand API. A Cloudflare Worker proxy to protect the API key is pending deployment. See Developer docs for integration details.
